# Stock Price Analysis and Prediction System

## 📊 Project Overview

A comprehensive data science project for stock price analysis and prediction using machine learning. This project demonstrates end-to-end data science workflows including data collection, preprocessing, exploratory data analysis, visualization, and predictive modeling.

## 🎯 Key Features

- **Data Collection**: Automatic fetching of real-time stock data from Yahoo Finance
- **Technical Indicators**: MACD, RSI, Bollinger Bands, Moving Averages
- **Exploratory Analysis**: Statistical analysis, correlation studies, risk metrics
- **Machine Learning Models**: Random Forest, Gradient Boosting, Linear Regression
- **Interactive Visualizations**: Candlestick charts, heatmaps, price predictions
- **Comprehensive Reporting**: Automated analysis reports and predictions
